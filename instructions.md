# Инструкция: shorts-generator

Краткая справка по боту. В Telegram то же самое открывается кнопкой **📖 Инструкция**.

## Что делает бот

1. Пишет сценарий Shorts (Gemini).
2. Озвучивает его (Edge TTS / ElevenLabs).
3. Собирает видео-заготовку: фон + голос (1080×1920).
4. По желанию готовит пост для Telegram-канала.

Субтитры, вставки и финальный монтаж — вручную в любом монтажёре.

## Главное меню

| Кнопка | Назначение |
|--------|------------|
| 🚀 Сгенерировать видео | Формат → свой промпт → сценарий + mp3 + mp4 |
| 💡 Придумать тему | Формат → диалог с ИИ → промпт → генерация |
| 🎙 Озвучить сценарий | Свой текст → только mp3 |
| 📝 Сделать пост | Пост на тему или разбор к ролику |
| 📖 Инструкция | Эта справка в чате |

На любом шаге кнопка **📱 Главное меню** сбрасывает сценарий и возвращает в начало.

## Форматы ролика

| Формат | О чём писать |
|--------|----------------|
| 📖 История | Герой, схема, цифры, чем кончилось (~45 сек) |
| 📰 Новость | Событие, смысл для зрителя, риски |
| 💡 Фишка / репо | Инструмент: зачем → как → результат (без биографий) |

## Как писать промпт (🚀 Сгенерировать видео)

Хороший промпт — одна конкретная идея, не «сделай что-нибудь про ИИ».

**История:** кто + что сделал + ключевая цифра.  
Пример: `Джон Дагита украл 46 млн у US Marshals — как схема работала и как его поймали`

**Новость:** факт + угол для аудитории.  
Пример: `OpenAI купила Windsurf — что изменится для разработчиков и Cursor`

**Фишка:** продукт + зачем смотреть.  
Пример: `Remotion: как собрать вертикальный Short из JSON без ручного монтажа с нуля`

Цифры в промпте можно арабскими — в сценарии бот переведёт словами для озвучки.

## Как пользоваться «Придумать тему» (💡)

1. Выбери формат.
2. Пиши коротко: ниша, запреты, примеры («крипто-истории, без политики»).
3. Когда идея сошлась — **📝 Сформировать промпт**.
4. Проверь текст, при необходимости поправь и отправь сообщением или жми **🚀 Сгенерировать видео**.

Не жми «Сформировать промпт» с пустым чатом — сначала хотя бы одно сообщение.

## Озвучка и правка

- **🎙 Озвучить** — фразы с новой строки, без ремарок в скобках.
- После ролика **✏️ Откорректировать сценарий** — вставь полный новый текст, бот переозвучит и пересоберёт mp4.

## Посты

- **📰 Пост на тему** — разбор новости/инсайта для канала.
- **🎬 Пост к ролику** — первая строка тема, дальше сценарий TikTok; в Telegram бот даёт полный разбор без отсылок к Shorts.

## Результат генерации

На выходе:

- `result.mp4` — фон + голос (заготовка)
- `voiceover.mp3` — дорожка для монтажа
- текст сценария — для субтитров

Дальше: импорт в монтажёр, субтитры по фразам, оверлеи из `assets/overlays/` (локально).

## Типичные ошибки

| Плохо | Лучше |
|-------|--------|
| «Тема про ИИ» | Конкретный кейс или релиз + угол |
| Смешать историю CEO в «Фишке» | Фишка = только инструмент |
| Промпт на полстраницы воды | 1–3 предложения с фактами |
| Ждать готовый TikTok из бота | Бот даёт заготовку; монтаж — отдельно |
